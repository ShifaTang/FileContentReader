# 1.读取txt文本文件

## 1.1 函数名: `read_txt`

**描述**:  一次性读取文本文件内容,适合小文件

**参数**:  `file_path:<&str>` : 文件路径

**返回值**:  `Result<String>` : 返回字符串,字符串包含整个txt文件的内容

**样例测试**:

- **正确情况测试**
 - **输入**: 文件路径
 - **输出**:
  ```
  Hello Rust!
  文件内容读取功能库！
  ```

+ **错误情况测试**
 - **错误类型1**:  文件路径不存在

  - **输入**:  不存在的txt文件路径
  
  - **输出**:  文件不存在,请检查路径是否正确
  
  - **实际错误**:  函数无法根据文件路径找到txt文件



 - **错误类型2**:  文件类型错误

  - **输入**:  传递非txt文本文件的文件路径
  
  - **输出**:  文件类型错误,请检查文件类型是否为txt文件 
  
  - **实际错误**:  函数根据文件路径判断文件非txt文本文件



 - **错误类型3**:  无法打开文件

  - **输入**:  传入没有访问权限的txt文件
  
  - **输出**:  拒绝访问。 (os error 5)
  
  - **实际错误**:  没有txt文件的访问权限,无法正常打开文件
